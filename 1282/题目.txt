1282. Reverse Vowels of a String
жпнд
English

Write a function that takes a string as input and reverse only the vowels of a string.
Example

Example 1:

Input : s = "hello"
Output :  "holle"

Example 2:

Input : s = "lintcode"
Output : "lentcodi".

Notice

The vowels does not include the letter "y".

557. Count Characters
жпнд
English

Count characters in a string. Return a hash map which key is character and value is the occurrency of this character.
Example

Example 1:

Input:
str = "abca"

Output:
{
  "a": 2,
  "b": 1,
  "c": 1
}

Example 2:

Input:
str = "ab"

Output:
{
  "a": 1,
  "b": 1
}
